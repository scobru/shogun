Come Funziona?

SHOGUN utilizza Gun.js come base per creare un sistema completamente decentralizzato di gestione wallet. Le chiavi private sono gestite localmente, mentre la sincronizzazione e lo storage avvengono in modo distribuito. Gli indirizzi stealth forniscono un ulteriore livello di privacy per le transazioni.

Tutte le operazioni critiche avvengono nel browser dell'utente. I dati sensibili non lasciano mai il dispositivo dell'utente e sono sempre crittografati end-to-end.

Quick Start

// Inizializza WalletManager
import { WalletManager, StorageType } from './src/WalletManager';

const manager = new WalletManager();

// Crea un account
try {
  await manager.createAccount('username', 'password');
  console.log('Account creato!');
} catch (error) {
  console.error('Errore:', error);
}

// Crea e salva un wallet
const gunKeyPair = manager.getCurrentUserKeyPair();
const { walletObj, entropy } = await WalletManager.createWalletObj(gunKeyPair);

// Salva sia su Gun che in localStorage
await manager.saveWallet(walletObj, pubKey, StorageType.BOTH);
